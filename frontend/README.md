# AI Command Center - Frontend

A React-based authentication application with Login, Register, and Forgot Password pages.

## Tech Stack

- **React** - UI library
- **Vite** - Build tool
- **Material UI** - UI component library
- **TypeScript** - Type safety
- **React Router** - Page routing
- **Axios** - HTTP client

## Setup Instructions

### Prerequisites

- Node.js v22+
- npm or yarn

### Installation

```bash
cd frontend
npm install
```

### Environment Variables

Create a `.env` file in the root directory:

```env
VITE_API_BASE_URL=http://localhost:5000
```

### Development

```bash
npm run dev
```

The app will start at `http://localhost:5173`

### Build

```bash
npm run build
```
